# Задача: Реализовать io.Reader-адаптер

Необходимо создать адаптер для интерфейса `io.Reader`, который будет оборачивать существующий `io.Reader` и подсчитывать количество прочитанных байтов и строк.

---

## Описание

Адаптер должен реализовывать интерфейс `io.Reader` и дополнительно предоставлять методы для получения статистики:
- количество прочитанных байтов
- количество прочитанных строк

---

## Требования

1. Адаптер должен быть реализован в пакете `io_reader_adapter`
2. Адаптер должен оборачивать существующий `io.Reader`
3. Адаптер должен реализовывать интерфейс `io.Reader`
4. Адаптер должен подсчитывать количество прочитанных байтов
5. Адаптер должен подсчитывать количество прочитанных строк
6. Адаптер должен предоставлять методы для получения статистики:
   - `BytesRead() int64` - количество прочитанных байтов
   - `LinesRead() int64` - количество прочитанных строк

---

## Пример использования

```go
file, err := os.Open("example.txt")
if err != nil {
    log.Fatal(err)
}
defer file.Close()

adapter := NewReaderAdapter(file)

buffer := make([]byte, 1024)
for {
    n, err := adapter.Read(buffer)
    if err != nil && err != io.EOF {
        log.Fatal(err)
    }
    if n == 0 {
        break
    }
    // Обработка данных
}

fmt.Printf("Прочитано байт: %d\n", adapter.BytesRead())
fmt.Printf("Прочитано строк: %d\n", adapter.LinesRead())
```

## Методы адаптера

1. **NewReaderAdapter(reader io.Reader) *ReaderAdapter** - создает новый адаптер
2. **Read(p []byte) (n int, err error)** - реализация интерфейса io.Reader
3. **BytesRead() int64** - возвращает количество прочитанных байтов
4. **LinesRead() int64** - возвращает количество прочитанных строк

---

## Требования к реализации

- Использовать только стандартные библиотеки Go
- Подсчет строк должен учитывать различные окончания строк (\n, \r\n)
- Адаптер должен быть потокобезопасным
