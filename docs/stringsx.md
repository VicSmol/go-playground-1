# Задача: Реализовать пакет `stringsx`

Необходимо создать пакет `stringsx` в Go, который будет предоставлять вспомогательные функции для работы со строками. В рамках задачи нужно реализовать три основные функции:

---

## 1. **Нормализация строк**

Функция должна выполнять следующие действия:
- Удаление лишних пробелов в начале и в конце строки.
- Замена нескольких пробелов (или других пробельных символов) в середине строки на один пробел.
- Приведение строки к нижнему регистру (или, в зависимости от требований, оставить как есть, если не указано иное).

Пример:
```go
Normalize("  Hello   World  ") // -> "Hello World"
```
---

## 2. **Безопасный split и join**
   
a) Безопасный Split
   Функция должна безопасно разбивать строку по разделителю:
   Обрабатывать случай, когда строка пустая или содержит только разделители.
   Не возвращать пустые строки в результате, если это не требуется.

Пример:

```go
Split("a;;b;c;", ";") // -> []string{"a", "b", "c"}
```
b) Безопасный Join
Функция должна объединять строки с использованием указанного разделителя:
Обрабатывать случай, когда передан пустой список строк.
Использовать безопасный способ соединения, чтобы избежать лишних разделителей.

Пример:

```go
Join([]string{"a", "b", "c"}, ",") // -> "a,b,c"
```

---

## 3. Парсинг строки вида key=value;key2=value2 ##
   Функция должна разбирать строку вида:

```go
key1=value1;key2=value2;key3=value3
```
и возвращать map[string]string, где ключи и значения — это значения из строки.
Пример:

```go
ParseKV("name=John;age=30;city=NYC") // -> map[string]string{"name": "John", "age": "30", "city": "NYC"}
```
---

## Требования: ##

- Пакет должен находиться в директории internal/stringsx.go.
- Необходимо реализовать только указанные функции.
- Не использовать внешние библиотеки, кроме стандартных библиотек Go (strings, strconv, unicode, и т.д.).

---

## Пример структуры пакета: ##

```go
package stringsx

func Normalize(s string) string
func Split(s, sep string) []string
func Join(a []string, sep string) string
func ParseKV(s string) map[string]string

```

Эти функции должны быть готовы к использованию в других частях проекта.
